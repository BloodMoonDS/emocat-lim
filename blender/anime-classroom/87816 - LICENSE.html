About "Anime Classroom":

  • Blender: 2.78
  • Render Engine: Freestyle
  • Uploaded on: 2017-03-12 23:12:03
This is a classroom scene based on the style of classrooms common in anime. I did not base it on any one particular anime classroom but took elements from a large number of reference images. In addition to the classroom there is also the hallway outside the room. I closed off the end of the hall but you could easily put multiple rooms together to make a longer hall. One idea I have is to make other types of rooms and potentially a whole school building but I started this scene over a year ago and only tend to do a little work on my models when I feel like it so it could be a long time to get something like that done. I also have a collection of other half finished models that I also want to finish.Modelling the desk object was the most complicated part. I also spent quite a bit of time getting the lighting to match the feel in the reference images that I was looking at. I would say that the biggest lesson I learnt from this project was that reference images can really help when it comes to lighting since I felt like I had a target that I was working toward.I made all of the models in this scene. All of the image textures are from photographs that I took. The background image that can be seen outside the window is a photo I took on a recent trip to Japan.Feel free to use this however you like. If you use it and mention where you got it that would be nice but is not required. I made these models as practice and am interested if you have feedback on how I can improve.
